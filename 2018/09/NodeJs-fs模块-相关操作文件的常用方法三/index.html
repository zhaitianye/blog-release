fs.chown(path, uid, gid, callback)

  • path <string> | |
  • uid <integer>
  • gid <integer>
  • callback
    • err

异步地更改文件的所有者和群组。 除了可能的异常，完成回调没有其他参数。 也可参阅 chown(2)。

fs.chownSync(path, uid, gid)

  • path <string> | |
  • uid <integer>
  • gid <integer>

同步地更改文件的所有者和群组。 返回 undefined。 这是 fs.chown() 的同步版本。 也可参阅 chown(2)。
