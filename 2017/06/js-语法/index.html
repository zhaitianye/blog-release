JavaScript 关键字

JavaScript 关键字用于标识要执行的操作。 和其他任何编程语言一样，JavaScript 保留了一些关键字为自己所用。 var 关键字告诉浏览器创建一个新的变量：

JavaScript 同样保留了一些关键字，这些关键字在当前的语言版本中并没有使用，但在以后 JavaScript 扩展中会用到。 JavaScript 关键字必须以字母、下划线（_）或美元符（$）开始。 后续的字符可以是字母、数字、下划线或美元符（数字是不允许作为首字符出现的，以便 JavaScript 可以轻易区分开关键字和数字）。

JavaScript 函数

JavaScript 语句可以写在函数内，函数可以重复引用： 引用一个函数 = 调用函数(执行函数内的语句)。

JavaScript 字母大小写

JavaScript 对大小写是敏感的。 当编写 JavaScript 语句时，请留意是否关闭大小写切换键。 函数 getElementById 与 getElementbyID 是不同的。 同样，变量 myVariable 与 MyVariable 也是不同的。

JavaScript 字符集

JavaScript 使用 Unicode 字符集。 Unicode 覆盖了所有的字符，包含标点等字符。
