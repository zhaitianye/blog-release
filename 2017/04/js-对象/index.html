对象属性

可以说 “JavaScript 对象是变量的容器”。 但是，我们通常认为 “JavaScript 对象是键值对的容器”。 键值对通常写法为 name : value (键与值以冒号分割)。 键值对在 JavaScript 对象通常称为 对象属性。 对象键值对的写法类似于：

  • PHP 中的关联数组
  • Python 中的字典
  • C 语言中的哈希表
  • Java 中的哈希映射
  • Ruby 和 Perl 中的哈希表
