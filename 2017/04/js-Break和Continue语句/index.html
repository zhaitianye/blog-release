break 和 continue 语句仅仅是能够跳出代码块的语句。 语法:

break labelname; continue labelname;

continue 语句（带有或不带标签引用）只能用在循环中。 break 语句（不带标签引用），只能用在循环或 switch 中。 通过标签引用，break 语句可用于跳出任何 JavaScript 代码块：
