jQuery 选择器

jQuery 选择器允许您对 HTML 元素组或单个元素进行操作。 jQuery 选择器基于元素的 id、类、类型、属性、属性值等”查找”（或选择）HTML 元素。 它基于已经存在的 CSS 选择器，除此之外，它还有一些自定义的选择器。 jQuery 中所有选择器都以美元符号开头：$()。

通过 $(“:button”) 可以选取所有 type=”button” 的 <input> 元素 和 <button> 元素，如果去掉冒号，$(“button”)只能获取 <button> 元素。

关于 : 和 [] 这两个符号的理解 ：可以理解为种类的意思，如：p:first，p 的种类为第一个。 [] 很自然的可以理解为属性的意思，如：[href] 选取带有 href 属性的元素。
