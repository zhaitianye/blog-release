启动MongoDB服务器

有关完整的MongoDB安装说明，请参阅手册。

  1. 从MongoDB下载正确的MongoDB版本
  2. 创建一个数据库目录（在本例中为/ data）。
  3. 安装并启动一个mongod过程。

mongod –dbpath = / data

您应该看到mongod进程启动并打印一些状态信息。
